# GaleriaHyperV
### Galería de Imágenes para Hyper‑V

Repositorio que centraliza una galería de imágenes de sistemas operativos para usar en Hyper‑V, gestionada mediante un archivo galeria.json y un script PowerShell para insertar o actualizar entradas.

## Contenido del repositorio
- galeria.json — Define cada imagen disponible: nombre, versión, parámetros y metadatos.

- insertar_galleria.ps1 — Script PowerShell para añadir o modificar elementos de la galería.

## Sistemas incluidos
Distribuciones y sistemas añadidos hasta ahora:

- Arch Linux

- CrunchBang++

- Devuan

- Fedora Silverblue

- GhostBSD

- Oracle Linux

- Pop!_OS

- Q4OS

- Qubes OS

- Solus

- Ubuntu

## Objetivo del proyecto
Proporcionar una colección estructurada y ampliable de imágenes para facilitar la creación y despliegue de máquinas virtuales en Hyper‑V.

## Estado
Proyecto en desarrollo activo con actualizaciones frecuentes.
